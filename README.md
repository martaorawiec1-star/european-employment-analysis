# European Employment Analysis (2015–2024)

This project analyzes employment trends across European countries using Eurostat data.

## Dataset
European employment and unemployment rates dataset from Kaggle.

## Tools
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis
The project explores:
- employment trends over time
- differences between countries
- gender disparities
- employment rates across age groups

## Key Insights
- Employment rates increased across Europe between 2015 and 2024.
- Northern European countries show the highest employment rates.
- Younger age groups have significantly lower employment levels.
